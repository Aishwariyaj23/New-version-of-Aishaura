## HOW TO USE THIS DATA

### Method 1: Copy-Paste into Google Sheets (Easiest)

1. **Create Products Sheet:**
   - Open Google Sheets
   - Select cell A1
   - Paste all data from PRODUCTS SHEET above

2. **Create Reviews Sheet:**
   - Add new sheet named "Reviews"
   - Select cell A1
   - Paste all data from REVIEWS SHEET above

### Method 2: Manual Entry (If you prefer)

Just follow the table structure above and enter each row manually.

## IMPORTANT NOTES

### Column Format
- **Product Name**: Must match exactly between sheets (case-sensitive)
- **Price**: Just numbers (no ‚Çπ symbol)
- **Image**: Path to image file (relative path in your project)
- **Rating**: 1-5 scale
- **Benefits & Usage**: Separate items with semicolons (;)
- **Date**: Any readable format ("2 weeks ago", "yesterday", etc.)

### Example of Missing Data Handling
- If a product has no reviews: Website will show "No reviews yet"
- If Review Count is missing: Uses count of actual reviews
- If Avg Rating missing: Calculates automatically from individual ratings
- If image is missing: Browser shows broken image icon

### Avoid These Mistakes

## FAQ

**Q: Can I have different review counts for the same product?**
A: No, the count should be consistent. Update it when you add new reviews.

**Q: What if I delete a review?**
A: Just delete the row from Reviews sheet. Website will update automatically.

**Q: Can I have products with no reviews?**
A: Yes! If a product has no reviews in the Reviews sheet, it will show "No reviews yet" on the website.

**Q: Do I need to redeploy the Apps Script?**
A: No! Just edit the Google Sheets directly. Website fetches data from sheets automatically.
